# CONTROLS
  ## MOVEMENT
    - W: forward
    - A: left
    - S: back
    - D: right
    - Q: down
    - E or SPACE: up
  ## LOOK
    - I: up
    - J: left
    - K: down
    - L: right
    - U: counter-clockwise
    - O: clockwise
