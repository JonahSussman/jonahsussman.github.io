Welcome to Tensorpedia!

From Tensorpedia, the poorly auto-generated encyclopedia

Welcome to Tensorpedia! Tensorpedia is a poorly auto-generated Wikipedia clone utilizing aitextgen and GPT-2. In other words, none of these articles exist! This website takes a given title and generates a viable-looking Wikipedia-like article about it, using Wikipedia's 10,000 "vital articles" as a dataset. If you want to learn more about how I created this project, check out my article here.

You can navigate to different pages within Tensorpedia in a few ways. The first and easiest way is to click on a link within the page. This will make a request to the server to either generate a new page, or show a previously generated cached version. You can also use the search box in the upper right-hand part of the page. Finally, you can generate a random page by clicking the random page button on the left.
